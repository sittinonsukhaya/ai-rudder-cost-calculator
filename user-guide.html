If the client has system data ‚Äî ask for their ACD/IVR report or phone bill. The total calls per month is the number you need.

If no system data is available ‚Äî use this formula:

Volume = Calls per agent per day × Working days per month × Number of agents
Example: 50 calls/day × 22 days × 100 agents = 110,000 calls/month

To estimate calls per agent per day, ask the client: "How many calls does one agent typically handle in a day?" If they don't know, use these industry benchmarks:

Type of Work Calls/Agent/Day Notes
Inbound customer service 25‚Äì50 Depends on call complexity; simple queries = higher end
Outbound sales (consultative/B2B) 40‚Äì80 Longer conversations, discovery calls
Outbound cold calling (manual dial) 80‚Äì150 Short calls, high rejection rate
Outbound with predictive dialer 150‚Äì250+ Dialer eliminates wait time between calls

Sources: LiveAgent 2025, Call Centre Helper, Sprinklr 2025

Quick tip: For most Thai contact centers doing outbound with manual dialing, 50‚Äì80 calls/agent/day is a safe starting point. With a predictive dialer, double or triple that number.

3e. ‡∏ä‡∏±‡πà‡∏ß‡πÇ‡∏°‡∏á Admin ‡∏ó‡∏µ‡πà‡∏õ‡∏£‡∏∞‡∏´‡∏¢‡∏±‡∏î/‡πÄ‡∏î‡∏∑‡∏≠‡∏ô ‚Äî ‡∏Ç‡∏∏‡∏°‡∏ó‡∏≠‡∏á‡∏ó‡∏µ‡πà‡∏ã‡πà‡∏≠‡∏ô‡∏≠‡∏¢‡∏π‡πà

This field captures the value of supervisory and administrative time that AI automation eliminates. Most clients underestimate this. Break it down into categories and add them up:

Hours/day per manager × Working days × Number of managers
Benchmark: 1‚Äì2 hrs/day. Example: 1.5 hrs × 22 days × 3 managers = 99 hrs/month
2. Report Generation & QA Review

Managers manually pull reports, review call recordings, and score agent performance. Industry data: managers currently review only 2‚Äì5% of calls manually. AI can auto-QA 100%.

Hours/day on reports/QA × Working days × Number of supervisors
Benchmark: 1.5‚Äì3 hrs/day. Example: 2 hrs × 22 days × 3 supervisors = 132 hrs/month

Prep hours per session × Sessions per agent × Number of agents / Agents per supervisor
Benchmark: 2‚Äì3 hrs prep per session. Example: With AI auto-analysis, prep drops from 2.5 hrs to 0.5 hrs, saving ~40 hrs/month per supervisor

Benchmark: 0.5‚Äì1 hr/day. Example: 1 hr × 22 days = 22 hrs/month
